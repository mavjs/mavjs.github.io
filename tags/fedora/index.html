World of Warcraft on Fedora 20 via wine

Install wine: # sudo yum install wine Download World of Warcraft Setup Installer from https://us.battle.net/account/download/?show=wow. Run the installer with wine: # cd ~/Downloads/ # wine World-of-Warcraft-Setup-enGB.exe After the installation completes, it will ask you to login to your Battlenet Account. Instead, quit the application. Then tell it to use software rendered OpenGL by executing (it has a bug where the interface is just black canvas on some intel cards): % export LIBGL_ALWAYS_SOFTWARE=1 % wine ~/.wine/drive_c/Program\ Files\ (x86)/Battle.net/Battle.net\ Launcher.exe Battlenet actually allows you to play even though it hasn’t finished downloading if it has reached a certain downloaded size.
